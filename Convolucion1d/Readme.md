Convolución 1d
==============

Implementación
--------------

Fueron implementadas 4 versiones de la convolución,
secuencial, paralela naive, palalela con caching y
paralela con caching y tecnica de tiling.

Datos
-----

Se utilizo un set de datos pequeño de potencias de 10,
de 10^4 hasta 10^8.
